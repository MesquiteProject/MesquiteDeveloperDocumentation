Strings

(August 2005)

(under construction)

Here listed are some of the utility classes related to Strings.

  • ParseUtil — an important class for parsing strings (tokens, etc.)
  • StringUtil — some string utilties
  • Parser — an important class for parsing strings (tokens, etc.)
  • StringInABox — draws string in a box (like TextArea but without border
  • MesquiteString - a wrapper class for strings
  • NameReference - a class that helps maintain a list of "registered" strings, allowing quick comparison for equality of strings when the query might be made many times
  • TextRotator - to draw rotated strings
