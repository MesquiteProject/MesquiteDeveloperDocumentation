Utilities & interfaces

(August 2005)

(under construction)

Here listed are some of the utility classes and interfaces of the Mesquite class libraries.

Utilities

  • Random numbers & probability distributions:
    • Binomial
    • RandomBetween
  • MesquiteBoolean -wrapper class for booleans
  • MesquiteTimer -simple class for profiling
  • MesquiteImage - for getting images
  • TextRotator - to draw rotated strings
  • Bits - for storing boolean array
  • ListableVector - Vector whose objects have a getName() method
  • StringDialog, ListDialog, MesquiteInteger.queryInteger, MesquiteDouble.queryDouble - dialogs & methods that can easily query to return strings, numbers, choice from a list, etc.

Special interfaces

  • Listable - has the getName() method
  • Associable - has parts to which auxiliary information can be attached (e.g., nodes of a tree)
  • HNode -has daughters and an ancestor, and thus can be displayed in a hierarchy browser (e.g, employee tree; project, files and file elements).
